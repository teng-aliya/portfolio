process

  1. define

    Define, discuss, and established the objectives your project. We can make rough outlines and carefully curate what layouts and components suit best with your project when discussing together. This discussion can last possibly for a couple of meetings, depending on the difficulty of the project at hand.

  2. design

    Once there's a confirmed set of objectives and themes, we can start designing the layout of the website. I will build multiple frameworks according to the themes and objectives that was discussed from the previous meetings we've had. That way, a variety of layouts can be selected by the client before fine-tuning begins.

  3. fine-tune

    Once a framework is chosen, another set of meetings will be conduct so fine-tuning can be made. Every detail of the chosen framework will be fine-tuned to the client's liking to ensure the website is within the ideals of the client.

  4. build

    Lastly, I'll build the website according to the final and official framework that has been approved by the client. A progress log can be sent to the client if upon requested. The time spent on building website depends on the difficulty of the website.
